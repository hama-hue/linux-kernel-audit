# Linux Kernel Audit Project

## Overview

This project is a structured audit of the Linux Kernel, focusing on its architecture, philosophy, and real-world impact. It also includes practical shell scripts demonstrating Linux system operations.

## Objective

* Understand open-source philosophy
* Explore Linux kernel structure
* Perform system-level operations using shell scripting

##Scripts Included

### 1. System Identity Script

Displays system information like hostname, kernel version, OS, and architecture.

### 2. Process Monitor Script

Shows top CPU-consuming processes using Linux commands.

### 3. Log Analyzer Script

Analyzes failed login attempts from system logs.

### 4. Backup Script

Creates backups of shell scripts automatically.

### 5. System Update Script

Checks for system updates using package manager.

##Environment

* OS: Ubuntu (WSL2)
* Shell: Bash

## Key Learnings

* Linux system commands
* Process and log management
* File handling and automation
* Open-source system interaction

## Conclusion

This project demonstrates practical understanding of Linux systems and open-source principles through hands-on scripting and analysis.
